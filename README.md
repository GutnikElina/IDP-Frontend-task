# Frontend Tasks

This repository contains frontend components for web applications, served statically.

## Implemented Features
- **Static Assets**: Contains a basic structure for static web application resources inside `src/main/resources/static/`.
- **HTML Structure**: `index.html` file providing the main entry point for the frontend.
- **Styling**: Vanilla CSS implemented in `style.css` for application styling.
- **JavaScript Logic**: Core frontend logic and interactions handled in `app.js`.
